Rule of thumb for each technique

  • Zero-shot — instruction only, no worked examples. You rely on the model's general training.
  • One-shot — exactly one example is shown to anchor the format, then the model continues.
  • Few-shot — two or more examples establish a pattern the model imitates.
  • Prompt chaining — the prompt explicitly consumes the output of a previous step and pushes it forward.
  • Meta-prompting — you ask the model to write or improve a prompt, not to solve the task directly.
